
write a blog on the difference between document & window objects

document : It represents any HTML document or web page that is loaded in the browser.
 It is loaded inside the window. It is the object of window property We can access the document from a window using the window. 
 document The document is part of BOM (browser object model) and dom (Document object model) syntax: document.propertyname;

window : It represents a browser window or frame that displays the contents of the webpage.
 It is the very first object that is loaded in the browser. It is the object of the browser. 
 We can access the window from the window only. i.e. window.window The window is part of BOM, not DOM. syntax: window.propertyname;
